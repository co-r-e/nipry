# Notes Style

Standard note format generated by `speaker-notes-polisher`.

## Base Structure

```text
Purpose:
- State the primary communication goal in one sentence.

Talking Points:
- Point 1
- Point 2
- Point 3

Estimated Time:
- 45 seconds
```

## Rules

- Keep section order: `Purpose` -> `Talking Points` -> `Estimated Time`.
- Use concise bullets that are easy to read aloud.
- Keep lines short enough for natural pacing while speaking.
- Express estimated time in seconds.
- Prefer concrete terms from slide content (headings/keywords), not abstract placeholders.
