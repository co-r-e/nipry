# TailwindCSS in Remotion

You can and should use TailwindCSS in Remotion, if TailwindCSS is installed in the project.

## Important restriction

Do not use Tailwind's built-in transition and animation utilities. All animations must be driven by `useCurrentFrame()`.

## Setup

For setup instructions, see: https://www.remotion.dev/docs/tailwind
